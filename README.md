# Dev Retos

**Dev Retos** es una plataforma móvil diseñada para que desarrolladores potencien sus habilidades mediante retos diarios, práctica constante y gamificación. La aplicación ofrece una experiencia premium con seguimiento de progreso, analíticas avanzadas y un sistema competitivo.

## 🌟 Características Principales

- **Retos Diarios:** Desafíos diarios para mantener una racha (*streak*) y ganar XP.
- **Modo Práctica:** Sesiones enfocadas en mejorar habilidades específicas.
- **Dashboard PRO:** Estadísticas detalladas, maestría de tecnologías y analíticas visuales.
- **Ranking Global:** Compite con otros desarrolladores y sube en la tabla de posiciones.
- **Gamificación:** Sistema de niveles, racha de días consecutivos y notificaciones inteligentes.
- **Offline First:** Soporte para funcionamiento básico sin conexión y sincronización fluida.

## 🛠️ Stack Tecnológico

- **Framework:** Flutter 3.10+
- **Estado:** Riverpod (Arquitectura reactiva con generación de código).
- **Navegación:** Go Router.
- **Persistencia:** libSQL / Turso (Base de datos distribuida de alto rendimiento).
- **Backend/Auth:** Firebase (Authentication & Cloud Services) y Google Sign-In.
- **UI/UX:** Flex Color Scheme (Temas oscuros premium), Google Fonts y Custom Painter para gráficos.
- **Monetización:** Google Mobile Ads (AdMob).

## 📂 Estructura del Proyecto

- `lib/core`: Configuración global, temas, rutas y clientes de base de datos.
- `lib/features`: Arquitectura orientada a funciones (Auth, Diario, Práctica, Pro, Ranking).
- `assets`: Configuración de entorno (.env), sonidos y recursos visuales.
